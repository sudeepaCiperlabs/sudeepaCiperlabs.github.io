Sinhala Data Corpus Editor

A language processing tool which, users can upload tagged Sinhala data corpus and search Sinhala words and ` tags within the corpus. Users can edit the tags for choosen words and update the corpus.

Gym Management System

A Management system for Gyms supported by a fingerprint authentication system. We built the fingerprint authentication system from scratch in a way it fits the client needs. Report generation, billing, Registration, automated notifications, flexibility of creating new package types. (Java EE, Jaspersoft, Arduino)

Unicode Converter

This project was originated by Sinhala Dictionary Office, Sri Lanka. One of the professors from university of Moratuwa was available as the mentor and the resource person. We developed an application where the users can upload a Docx file containing Sinhala/Tamil non-Unicode fonts. Then we will process the document and convert non-Unicode fonts to Unicode fonts and send the output as a Docx file. This application was a much needed one as many of the old government documents are written in non-Unicode fronts. For the second phase going to improve the system for PDF files. The prototype can be found here(UI are in underdevelopment) Technologies – NodeJs, HTML

TradeStation API integration

The project about integrating TradeStation (an online securities & futures brokerage firm) API with a trade signal provider’s twitter account as order placing can be automated. The system has two main parts. One for continually looking for a new tweet (PHP application). The other one is for placing the order after processing the data in the tweet! Technologies – NodeJs, ReactJs, PHP, MongoDB, AWS Lambda.
